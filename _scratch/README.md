# WhaleTrader — Development Scratch Pad

This directory contains archived experiments, debug scripts, and old benchmark versions.

These files are kept for reference but are NOT part of the active codebase.

## What's here

- `_debug_*.py` — Individual scenario debug scripts
- `_diag_*.py` — Diagnostic analysis scripts
- `_trace_*.py` — Signal trace tools
- `benchmark_v5/v6/v8/v9.py` — Old benchmark versions
- `backtester_v2-v9.py` — Old backtester versions
- `signal_engine_v5-v9.py` — Old signal engine versions

## Why keep them?

- Historical reference for what we tried
- Some diagnostic tools are still useful for debugging
- Shows the full iteration history (18 experiments, 7 succeeded)
