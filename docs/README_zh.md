### 信号引擎 — 6 因子模型

| 因子 | 权重 | 说明 |
|------|------|------|
| 动量 | 25% | 多时间框架 (5/10/20/50 bar)，波动率标准化 |
| EMA排列 | 25% | 5条EMA堆叠确认 (5/8/10/21/50) |
| RSI | 15% | 趋势上下文RSI，超买/超卖区间 |
| 突破 | 15% | 20 bar Donchian 通道突破 |
| 成交量 | 10% | 相对于20日均量的放量确认 |
| 布林带 | 10% | 均值回归确认 |

### 市场状态检测 — 7 种 Regime

| 状态 | 最大仓位 | 策略 |
|------|---------|------|
| 🔴 CRASH | 0% | 紧急退出 |
| 🟠 STRONG_BEAR | 10% | 仅防守反弹交易 |
| 🟡 BEAR | 15% | 小仓位逆势交易 |
| ⚪ RANGING | 45-68% | 均值回归 + 反锯齿 |
| 🔵 VOLATILE | 65% | 方向依赖，谨慎 |
| 🟢 BULL | 80% | 趋势跟随 |
| 🟢 STRONG_BULL | 92% | 最大信心 |

## 🔑 核心创新

1. **下降通道保护**: 30日下跌>10%时阻止牛市入场 (+1.68% alpha)
2. **连续亏损冷却**: 连亏后延长等待期，防止锯齿行情反复止损
3. **热手仓位调整**: 连赢时增大仓位，连亏时缩小
4. **Regime自适应**: 止损、仓位、入场阈值全部随市场状态调整
