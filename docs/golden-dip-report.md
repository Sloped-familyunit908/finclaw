## 1. 策略概述

**核心思路**: 确认大牛股 → 等回调黄金坑 → 反人性买入 → 坚定持有

### 买入条件
- 120日 R² > 0.6 (清晰上升趋势)
- 120日斜率 > 0 (方向向上)
- 60日收益率 > 20% (确实在涨)
- 从近期高点回调 > 10% (跌出黄金坑)
- RSI < 35.0 (短期超卖)
- 120日 R² > 0.5 (大趋势没破)
- 量能萎缩 < 70% 均量 (恐慌抛售结束)

### 仓位管理
- 首次买入: 30% 仓位
- 继续下跌 5%: 加仓到 50%
- 再跌 5%: 满仓 100%

### 卖出条件 (反人性：不轻易卖)
- 120日 R² < 0.3 (趋势真的破了)
- 从最高点回撤 > 25% (止损保护)
- RSI > 90.0 连续 3 天 (极度过热)

### 策略定位分析

- **cn_scanner短线**: 高频交易，每天需要盯盘，适合专业交易者

- **黄金坑买入**: 介于两者之间，在确认趋势的基础上等待黄金坑买入，
  既避免了追高，又不需要每天操作。核心优势是**买入时机更好**，
  通过等待回调来获得更好的风险收益比。
