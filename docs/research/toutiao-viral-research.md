### 2.1 冷启动算法机制（综合多篇资深创作者实战文章）

头条推荐算法的核心流程：
```
发布 → 初始推荐池（200-500人）→ 评估CTR+完读率+互动
    → 通过 → 扩大推荐池（2000-5000人）→ 再评估
    → 通过 → 爆发推荐（10万-100万+）
    → 未通过 → 停止推荐
```

**冷启动期关键数据**（来源：腾讯云开发者社区文章 2025-10）：
- 冷启动时间窗口：**30-60分钟**

- 阅读量波动可减少63%（通过时间因子优化）

### 2.2 新号破万阅读的实战策略

**来源：头条号实操文章 + 知乎13个冷启动账号操盘手分享**

#### 账号定位（前3天）
1. **完善资料**：头像用真人或专业logo，简介写清定位
2. **选择垂直领域**：不要杂，专注1个细分（如"A股周复盘"）
3. **前10篇文章决定账号生死** — 算法会标记新号潜力

#### 标题公式（实测最有效）
- **数字+悬念+痛点词**：`90%的人不知道！退休金这样领，每月多拿2000`
- **标题长度**：22-30字最佳
- **禁忌**：新号前3个月不要照搬爆文标题，会被判搬运，直接废号！
- **新号建议**：标题中规中矩但信息量大，不要过度标题党

#### 发布时间（黄金时段）
| 时段 | 流量等级 | 说明 |
|------|---------|------|
| 05:30-07:30 | ⭐⭐⭐⭐⭐ | 早起刷手机，中老年用户高峰 |
| 11:30-13:00 | ⭐⭐⭐ | 午休时间 |

| **周末特殊** | ⭐⭐⭐⭐ | 全天流量分布更均匀，10:00-12:00也不错 |

**⚠️ "自杀式发文时段"**（知乎操盘手透露）：凌晨0-5点发文，推荐量骤降。

#### 图片规则
- **每300字配一张图**
- **3张图**是最佳配图数量（三图排列封面最吸引人）
- 用头条自带图库或Canva免费素材，避免版权问题
- **横图优于竖图**（信息流展示面积更大）

#### 文章长度
- **最佳字数：800-1500字**
- 太短（<500）信息量不足，不推荐
- 太长（>2000）完读率下降
- **段落≤3行**，保持移动端阅读体验

### 2.3 微头条 vs 文章：新号该选哪个？

| 维度 | 微头条 | 文章（图文） |
|------|--------|-------------|
| 字数 | 50-2000字 | 800-5000字 |
| 爆款概率 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| 收益门槛 | 100粉丝即可 | 100粉丝 |
| 推荐机制 | 更宽松，易被推荐 | 竞争更大 |
| 单篇展现量 | 微头条爆款案例：121.6万展现，14.2万阅读 | 文章爆款更难但天花板更高 |
| **新号建议** | ✅ 优先选择！门槛低，容易起量 | 后期（1000粉后）逐步增加 |

**结论：新号应以微头条为主，文章为辅。微头条更容易获得推荐和互动。**

### 2.4 互动引导技巧（算法核心指标）

算法最看重的指标排序：
1. **点击率（CTR）**：标题+封面决定
2. **完读率**：内容结构+长度决定
3. **评论量**：在文中埋"争议点"引发讨论
4. **点赞/收藏**：文末引导 "觉得有用点个赞"
5. **转发量**：触发情感共鸣

**互动钩子设计**：
- 文中埋争议点："都说睡前喝酒助眠？邻居试了半年，医生直摇头…"
- 结尾问句："你遇到过类似的情况吗？"
- 首小时互动：请家人朋友点赞评论，帮助"暖"推荐
