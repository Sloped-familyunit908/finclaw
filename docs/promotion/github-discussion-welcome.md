# Welcome to FinClaw! Introduce yourself and share your use case

Thanks for checking out FinClaw.

FinClaw is an open-source quantitative finance engine that uses genetic algorithms to evolve trading strategies across 41 factor dimensions. It supports US stocks, Chinese A-shares, and cryptocurrency markets, and ships with 10 MCP tools for AI-native integration.

Key capabilities:

- Self-evolving strategy engine (genetic algorithm, walk-forward validated)
- Real-time quotes and historical data from free data sources
- Technical indicators, stock screening, portfolio analysis
- Strategy backtesting with Monte Carlo simulation
- MCP server for Claude, Cursor, and other AI assistants
- Production dashboard with TradingView charts
