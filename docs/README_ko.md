## 🏗️ 아키텍처

- **시그널 엔진**: 6팩터 모델 (모멘텀, EMA, RSI, 돌파, 거래량, 볼린저밴드)
- **레짐 감지**: 7가지 시장 상태 자동 식별 (크래시 ~ 강세장)
- **포지션 관리**: 트레일링 스톱, 피라미딩, 손익분기 스톱
- **리스크 관리**: 레짐 적응형 손절/이익실현
- **테스트**: 34개 회귀 테스트
