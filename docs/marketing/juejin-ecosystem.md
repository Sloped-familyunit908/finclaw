# 一个人+一个AI，如何在10天内构建3个开源项目（9000+测试，3种语言）

从3月17日到3月27日，我和我的 AI 助手螃蟹构建了 3 个开源项目。

不是 Demo。不是 PoC。是有完整测试、文档、CI/CD 的正经项目。总共跑通了 **9000 多个测试**。

我来复盘一下这10天到底发生了什么。

## 三个项目

### 1. FinClaw —— AI 量化交易引擎

**语言**：Python  
**测试数量**：5600+  
**干什么的**：用遗传算法自动进化交易策略

传统量化需要你自己想策略、写策略、调参数。FinClaw 的思路不一样——你告诉它目标（收益率、夏普比率、最大回撤），它自己进化出策略。

核心技术栈：
- 遗传算法引擎：支持 484 维因子空间
- Walk-Forward 验证：对抗过拟合
- 数据层：对接多个行情源（加密货币、A股）
- MCP Server：可以接入 AI Agent 做智能分析

GitHub: [github.com/NeuZhou/finclaw](https://github.com/NeuZhou/finclaw)

### 2. AgentProbe —— Playwright for AI Agents

**语言**：TypeScript  
**测试数量**：2907  
**干什么的**：给 AI Agent 写端到端测试

Playwright 让你测试网页应用的行为，AgentProbe 让你测试 AI Agent 的行为。

录制 Agent 的工具调用序列，回放验证一致性，支持语义级别的断言。做 Agent 的团队终于可以有正经的 CI/CD 流水线了。

GitHub: [github.com/NeuZhou/agentprobe](https://github.com/NeuZhou/agentprobe)

### 3. ClawGuard —— AI Agent 免疫系统

**语言**：TypeScript  
**测试数量**：684  
**干什么的**：检测和防御 AI Agent 面临的安全威胁

285+ 种威胁检测模式，覆盖 OWASP LLM Top 10。零依赖，纯 TypeScript。能做 Prompt 注入检测、PII 信息脱敏、工具调用审计。

GitHub: [github.com/NeuZhou/clawguard](https://github.com/NeuZhou/clawguard)

## 10天怎么做到的

坦白讲，如果没有 AI 辅助开发，这个产出不可能。

我的开发模式是这样的：

**白天**：我负责架构设计、需求定义、代码 Review、关键决策。
**全天候**：螃蟹（我的 AI 助手）负责编码实现、测试编写、文档生成、bug 修复。

具体的工作分配：

| 环节 | 人 | AI |
|------|----|----|
| 产品方向 | ✅ | |
| 架构设计 | ✅ | 参考建议 |
| 编码实现 | Review | ✅ |
| 测试编写 | 定义用例 | ✅ |
| Bug 修复 | 定位 | ✅ |
| 文档 | 审核 | ✅ |
| Git 管理 | | ✅ |

几个关键经验：

### 1. AI 不是万能的，但它是个极好的执行者

我不会让 AI 替我做架构决策。"用遗传算法还是强化学习""安全检测做规则引擎还是 ML""测试框架用录制回放还是声明式"——这些决策是我做的。

但一旦决策定了，AI 的执行速度是人类的 10 倍以上。它不需要休息，不会走神，能同时记住整个代码库的上下文。

### 2. 测试数量不是刻意追求的

9000 多个测试不是"为了数字好看"刻意凑的。这是 TDD 开发模式的自然结果——每个功能先写测试再写实现，测试数量就是这样堆起来的。

AI 写测试的速度很快。你告诉它"这个函数需要覆盖这几种边界情况"，它几分钟就写完了。换人来写，可能要半小时。

### 3. 真正的瓶颈是方向，不是速度

10 天里最慢的时刻不是写代码的时候，而是我在想"这个东西到底应该怎么设计"的时候。

AI 可以一小时写完 500 行代码。但如果方向错了，这 500 行全是浪费。

最有价值的人类贡献是：**想清楚要做什么**。

### 4. 三种语言不是问题

FinClaw 用 Python（量化金融生态决定的），AgentProbe 和 ClawGuard 用 TypeScript（前端/Node.js 生态决定的）。

对 AI 来说，切换语言几乎没有成本。对人来说，切换上下文是有成本的。所以我通常一个时间段只关注一个项目，让 AI 在另两个项目上持续推进。

## 写在最后

这不是在宣传"AI 取代程序员"。恰恰相反——这 10 天让我更清楚地认识到：**人类的判断力和 AI 的执行力是互补的。**

AI 不会替你想清楚一个产品应该长什么样。但一旦你想清楚了，它能帮你以极快的速度把它变成现实。
