# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 5.x.x   | ✅                |
| < 5.0   | ❌                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **DO NOT** open a public issue
2. Email: [security@neuzhou.dev](mailto:security@neuzhou.dev) or open a [private security advisory](https://github.com/NeuZhou/finclaw/security/advisories/new)
3. Include: description, steps to reproduce, potential impact
4. We will respond within 48 hours

## Security Measures

FinClaw never stores API keys in code. All credentials use environment variables.
